# Project

E-Commerce Sales Data Analysis 📊
Overview

This project focuses on analyzing E-Commerce sales data to uncover business insights, customer purchasing behavior, sales trends, and product performance using Python and data analytics techniques.

The analysis includes:

Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Sales trend visualization
Customer and product analysis
Business insights generation
Dataset

The dataset contains E-Commerce sales records including:

Order details
Product categories
Customer information
Sales amount
Profit and quantity data
Shipping details

Files used:

ecommers sales.csv
cleaned_ecommerce_sales.xls
Technologies Used 🛠️
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Project Workflow 🚀
1. Data Collection

Imported raw E-Commerce sales dataset.

2. Data Cleaning
Removed missing values
Handled duplicates
Corrected data types
Cleaned inconsistent entries
3. Exploratory Data Analysis (EDA)

Performed analysis on:

Sales distribution
Profit analysis
Top-selling products
Customer purchasing trends
Regional performance
4. Data Visualization

Created charts and graphs such as:

Bar charts
Pie charts
Line graphs
Heatmaps
5. Insights Generation

Generated meaningful business insights for decision-making.

Key Insights 📈
Identified top-performing product categories
Found high-revenue regions
Analyzed customer buying patterns
Detected seasonal sales trends
Compared profit across categories
